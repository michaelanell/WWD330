JavaScript: Novice To Ninja - Sitepoint

Key notes from chapters 5-7

  • Chapter 5: Objects


    • JavaScript objects are much like any other oop object, at least the basics.
    • To create an object:
      • const objectName = {};
      • Shorthand method of creating an object if the 'a property key is the same as a variable name that the property value is assigned to':
        • const name = 'Jane Austen';
          const realName = 'Jane Austen';
          // long way
          const author = { name: name, realName: realName };
          // short ES6 way
          const author = { name, realName };
      • To access a property use the . notation - author.name
      • To call a method - author.write()
      • Many built in objects available

  • Chapter 6 - DOM


    • The Document Object Model
      • The DOM is a network of nodes on a webpage.
      • HTML tag is the root node and everything else is a child of it.
      • DOM provides several ways to access any page element with the following methods that return an array Like node object:
        • objectName = document.elementYouWantToAccess;
        • typeOf objectName;
        • objectName.nodeType;
        • objectName.nodeName;
        • Document.body - returns body element of a web page
        • Document.images - returns node list of all images contained in document
        • Document.links - returns node list of all the a tag or area elements in document that have the href attribute
        • Document.anchors - returns a node list of all a tag elements that have the name attribute
        • Document.forms - returns a node list of all forms in the document
        • getElementById()
        • getElementsByTagName()
        • getElementsByClassName()
        • document.querySelector() - use CSS notation to find first element in document that matches that matches the CSS selector provided as an argument
        • document.querySelectorAll() - CSS notation returns node list of all elements in document that match CSS query selector
