The Window Object

  • Represented by the broswer window contains a web page.
  • There is a global object in every JS enviornment.
  • Variables created in global scope are properties of the global object.
  • In a browser enviornment the global object is the window object.
  • The Browser Object Model
    • Collection of properties and methods that have information about the bowser and computer screen.
    • No offical standard, but there are properties and methods supported by all major browsers, available through the window object.
  • Avoid using the window object to refer to global variables, as it makes code better portable between JS eviornments.
  • alert(), confirm(), and prompt() are all methods of the window object.
  • confirm() - 'will stop the execution of the program and display a confirmation dialog that shows the message provided as an argument, and giving the options of OK or Cancel.'
  • window.prompt() - will stop the execution of the program. It displays a dialog that shows a message provided as an argument, as well as an input field that allows the user to enter text. This text is then returned as a string when the user clicks OK.
  • eg. window.prompt('Please enter your name:');

    • Browser Information

  • Navigator object - 'contains information about the browser being used.'
  • userAgent property - returns information about the browser and operating system that is being used.
  • This information is not reliable since it can be modified by a user to show it as being a browser it is not.
  • window.location property - object that contains info about the url of the current page.
    Other properties:
    • href property - returns the full URL as a string (can be changed by assignment.)
    • protocol property - returns a string describing the protocol used (such as http, https, pop2, ftp etc.)
    • host property - returns a string describing the domain of the current URL and the port number.
    • hostname property - returns a string describing the domain of the current URL
    • port property - returns a string describing the port number
    • pathname property - returns a string of the path that follows the domain
    • search property - returns a string that starts with a “?” followed by the query string parameters
    • hash property - returns a string that starts with a “#” followed by the fragment identifier.
    • origin property - returns a string that shows the protocol and domain where the current page originated from.
    • reload() method - can be used to force a reload of the current page. If it’s given a parameter of true, it will force the browser to reload the page from the server, instead of using a cached page.
    • assign() method - can be used to load another resource from a URL provided as a parameter, for example
    • replace() method - similar to the assign() method, except the current page will not be stored in the session history, so the user will be unable to navigate back to it using the back button
    • toString() method - returns a string containing the whole URL

    Browser History

    • window.history property - accesses information about any previously visited pages in the current browser session.
    • window.history.length property - shows how many pages have been visited before arriving at the current page
    • window.history.go() method - used to go to a specific page (where 0 is the current page).
    • window.history.forward() - navigate forwards by one page.
    • window.history.back() - navigate backwards by one page.

    Screen Information

    • window.screen object contains information about the screen the browser is displayed on.
    • window.screen.height - returns height of the screen in pixels.
    • window.screen.width - returns width of the screen in pixels.
    • colorDepth property - used to find the color bit depth of the user’s monitor.

    The Document Object

    • The document object, an object of the window object, has properties and methods that deal with the page that has been loaded into the window. These include:
    • document.write() - the write() method writes a string of text to the page.

    Cookies

    • Small files saved locally on user's computer.
    • Stores information that can then be retrieved between requests
    • Can only be read by a web page from the same domain that set them.
    • Cookies can be used for personalizing a user’s browsing experience, storing user preferences, keeping track of user choices (such as a shopping cart), authentication and tracking users.
